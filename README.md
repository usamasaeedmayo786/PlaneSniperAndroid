# Plane Sniper Android

A 3D sniper jet combat game developed in Unity for Android platform.

## Description

Plane Sniper Android is an action-packed air combat game where players engage in intense aerial battles. The game features realistic aircraft models, dynamic combat mechanics, and immersive gameplay.

## Unity Version

- Unity 2021.3.43f1

## Features

- 3D aircraft combat gameplay
- Multiple aircraft models and assets
- Sniper-style targeting mechanics
- Realistic flight physics
- Visual effects and muzzle flashes

## Project Structure

- `Assets/` - All game assets including scripts, models, textures, and prefabs
- `ProjectSettings/` - Unity project configuration files
- `Packages/` - Unity package dependencies

## Getting Started

1. Clone this repository
2. Open the project in Unity 2021.3.43f1 or later
3. Open the main scene and press Play

## Build Instructions

1. Open Unity and load the project
2. Go to File > Build Settings
3. Select Android as the platform
4. Configure build settings and click Build
